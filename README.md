**PHKAPA** is a free, open source, CAPA management system for PHP, released under [MIT License]

It is powered by [CakePHP](http://cakephp.org) MVC framework.


## Requirements
  * Apache 2 with `mod_rewrite` or similar
  * PHP 5.2 or higher
  * MySQL 5 or higher
